# Trioxide

Svelte 5 customizable componets, focused on non-trivial UI pieces that are tedious to reimplement.

## Main focus

- Non-trivial UI pieces
- First-class accessibility
- First-class LTR/RTL support
- Fully customizable look
- Small footprint

## Status

- Work in progress; APIs may shift before a stable release.

## Components

- ContextMenu
- Docker
- Swappable
- Kbd
