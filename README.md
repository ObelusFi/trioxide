<h1 align="center">Trioxide</h1>

Svelte 5 customizable components, focused on non-trivial UI pieces that are tedious to reimplement.

## Install

```shell
npm i @obelusfi/trioxide
```

## Main focus

- Non-trivial UI pieces
- First-class accessibility
- First-class LTR/RTL support
- Fully customizable look
- Small footprint

## Status

- Work in progress; APIs may shift before a stable release.

## Components

- ContextMenu
- Docker
- Swappable
- Kbd
